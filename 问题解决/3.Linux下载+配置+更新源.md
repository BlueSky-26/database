### 2、Ubantu环境安装及配置源

```
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
sudo gedit /etc/apt/sources.list

删除（里面的文本）
复制（下面的源复制任一个）
粘贴并保存
点"X"关闭文件

sudo apt-get update
sudo apt-get upgrade

input>>tree -L 1 /
若出现树形目录结构，说明环境配置成功
```
